Equipa de 2-10 pessoas. MRR recorrente de clientes em retainer. Entregas mensais previsíveis: content calendar, ads, reports, copy. O desafio é escalar sem perder personalização.

💰 Custo real (Anthropic Pro/Max)

O OS é grátis (MIT). O Claude não é. Antes de instalar, sabe quanto vai custar para que funcione bem.

Componente Custo Quando se paga
iAmasters OS (este repo)Grátis (MIT)Nunca
Claude Desktop / CodeGrátis (app)Nunca
Anthropic Pro$20/mêsMínimo necessário para uso real
Anthropic Max$100-200/mêsSe vais usar muito Code intensivo
Firecrawl API (opcional)500 créditos grátisPara auto-extrair brand voice

⚠️ Plano Free não chega

Com Anthropic Free, os modelos bons (Opus, Sonnet 4+) não estão disponíveis. O OS funciona tecnicamente mas os outputs ficam fracos. Mínimo real: Anthropic Pro $20/mês.

💡 Pro ou Max?

Começa com Pro $20/mês. Se chegas ao limite de uso semanal regularmente (mensagens "limit reached"), passa a Max. Não há ponto em pagar Max se ainda não estoiraste o Pro.

6

🔧 Prerequisitos técnicos

O install.sh valida tudo antes de correr. Mas sabe à partida o que precisas — o erro #1 é Python da Microsoft Store em Windows.

📋 Prerequisitos

  • Claude Code CLI (incluído no Claude Desktop) — claude.ai/code
  • Node.js ≥18 — nodejs.org
  • Python ≥3.9 (para hooks Sinapsis) — python.org (NÃO Microsoft Store)
  • Git — provavelmente já tens
  • OS: macOS, Linux, Windows (Git Bash ou WSL)

⚠️ Windows: cuidado com Python

Se em Windows correres python e a Microsoft Store abrir, esse Python não serve. Instala de python.org e adiciona ao PATH. O installer deteta isto e bloqueia.

💡 Alternativa Windows

Se Windows nativo te dá dor de cabeça, usa WSL2 com Ubuntu. Experiência mais próxima de macOS/Linux, sem fricção de PATH/Microsoft Store.
